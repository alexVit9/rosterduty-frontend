# 📥 КАК СКАЧАТЬ ПРОЕКТ - Быстрый ответ

## ❌ В Figma Make НЕТ кнопки "Download"

Есть только:
- **Publish** - публикация приложения в интернет
- **Share** - поделиться проектом

---

## ✅ КАК ПОЛУЧИТЬ КОД - 3 варианта

### 🥇 Вариант 1: Через GitHub (ЛУЧШИЙ)

**Шаг 1:** Нажмите кнопку **"Share"** в Figma Make

**Шаг 2:** Найдите опцию с GitHub:
- "Connect to GitHub"
- "Export to GitHub"  
- "Deploy" → выберите GitHub

**Шаг 3:** Создайте репозиторий

**Шаг 4:** На вашем компьютере:
```bash
git clone https://github.com/ваш-username/проект.git
cd проект
npm install
npm run dev
```

**📖 Подробнее:** см. файл `GITHUB_EXPORT_GUIDE.md`

---

### 🥈 Вариант 2: Скопировать файлы вручную

**Я могу:**
1. ✅ Вывести содержимое каждого файла
2. ✅ Вы создадите файлы у себя локально
3. ✅ Скопируете содержимое

**Сколько файлов:** ~25 основных файлов вашего кода

**Время:** 30-60 минут

**Напишите:** "Хочу скопировать файлы вручную" и я создам полный список

---

### 🥉 Вариант 3: Создать проект с нуля

**Шаг 1:** Создайте Vite проект
```bash
npm create vite@latest crew-table-app -- --template react-ts
cd crew-table-app
npm install
```

**Шаг 2:** Установите зависимости
```bash
npm install react-router lucide-react sonner
```

**Шаг 3:** Я дам вам код каждого файла для копирования

**Время:** 45-90 минут
