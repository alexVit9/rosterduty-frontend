# 📥 Инструкция по скачиванию проекта

## 🎯 Варианты получения кода

В Figma Make нет кнопки "Download", но есть несколько способов получить код:

---

## ✅ Вариант 1: Через Share → Deploy (Рекомендуется)

### Что делает кнопка "Publish":
- Публикует приложение в интернет
- Даёт вам ссылку на работающее приложение
- НО не даёт исходный код

### Что делает кнопка "Share":
- Даёт доступ другим пользователям Figma Make
- Позволяет экспортировать на Netlify, Vercel и др.

### 🔧 Как получить код через Share:

1. **Нажмите "Share"** в верхнем меню
2. **Выберите "Deploy to..."** (если есть опция)
3. **Выберите платформу:**
   - **GitHub** - создаст репозиторий с кодом ✅ (лучший вариант)
   - **Netlify** - задеплоит и даст доступ к коду
   - **Vercel** - задеплоит и даст доступ к коду

4. После деплоя в GitHub вы сможете:
```bash
git clone https://github.com/ваш-username/проект.git
cd проект
npm install
npm run dev
```

## ✅ Вариант 2: Скопировать файлы вручную

Я создам для вас **ZIP-архив инструкцию** со всеми файлами.

### Шаг 1: Создайте папку на компьютере

```bash
mkdir crew-table-app
cd crew-table-app
```

### Шаг 2: Скопируйте все файлы

Я создам файл `PROJECT_FILES.md` со списком всех файлов и их содержимым.

Вы сможете:
1. Открыть каждый файл в списке
2. Скопировать содержимое
3. Создать файл у себя локально
4. Вставить содержимое

## 📦 Что нужно для запуска проекта локально

После получения кода вам нужно:

### 1. Установить Node.js
Скачать с https://nodejs.org/ (версия 18 или выше)

### 2. Установить зависимости
```bash
npm install
```

### 3. Создать .env файл
```env
VITE_API_URL=http://localhost:8000
```

### 4. Запустить проект
```bash
npm run dev
```
