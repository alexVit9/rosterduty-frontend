# 🐙 Экспорт проекта через GitHub (Лучший способ!)

## ✅ Почему GitHub - лучший вариант?

- ✅ Один клик - весь код в репозитории
- ✅ История изменений
- ✅ Легко клонировать на любой компьютер
- ✅ Можно продолжить разработку локально
- ✅ Бесплатно

---

## 🚀 Инструкция по экспорту в GitHub

### Шаг 1: Проверьте кнопку "Share"

1. **Найдите кнопку "Share"** в верхнем меню Figma Make
2. **Нажмите на неё**
3. **Ищите опции:**
   - "Connect to GitHub"
   - "Export to GitHub"
   - "Deploy to GitHub"
   - "Sync with GitHub"
   - Или просто иконку GitHub 🐙

### Если есть опция GitHub - отлично! Переходите к Шагу 2.

### Если НЕТ опции GitHub - используйте "Вариант Б" ниже.

---

## 📝 Вариант А: Есть интеграция с GitHub

### Шаг 2: Подключите GitHub

1. **Нажмите "Connect to GitHub"**
2. **Авторизуйтесь** в GitHub (если нужно)
3. **Разрешите доступ** Figma Make к вашему GitHub

### Шаг 3: Создайте репозиторий

1. **Введите название:** `crew-table-app`
2. **Выберите тип:** Public или Private
3. **Нажмите "Create" или "Export"**

### Шаг 4: Дождитесь загрузки

Figma Make автоматически:
- ✅ Создаст репозиторий на GitHub
- ✅ Загрузит все файлы проекта
- ✅ Создаст `.gitignore`
- ✅ Сделает первый коммит

### Шаг 5: Клонируйте на компьютер

```bash
# Скопируйте URL репозитория из GitHub
# Например: https://github.com/ваш-username/crew-table-app.git

# Клонируйте репозиторий
git clone https://github.com/ваш-username/crew-table-app.git

# Перейдите в папку
cd crew-table-app

# Установите зависимости
npm install

# Создайте .env файл
echo "VITE_API_URL=http://localhost:8000" > .env

# Запустите проект
npm run dev
```

🎉 **Готово! Проект работает локально!**

## 📝 Вариант Б: НЕТ интеграции с GitHub

Если в Figma Make нет прямой интеграции с GitHub, создадим репозиторий вручную:

### Шаг 1: Создайте репозиторий на GitHub

1. Откройте https://github.com
2. Войдите в аккаунт
3. Нажмите **"New repository"** (зеленая кнопка)
4. **Название:** `crew-table-app`
5. **Описание:** "Restaurant checklist management app"
6. **Тип:** Public или Private
7. ✅ Отметьте "Add a README file"
8. ✅ Отметьте "Add .gitignore" → выберите "Node"
9. Нажмите **"Create repository"**

### Шаг 2: Клонируйте пустой репозиторий

```bash
git clone https://github.com/ваш-username/crew-table-app.git
cd crew-table-app
```

### Шаг 3: Создайте Vite проект

```bash
# Инициализировать Vite проект в текущей папке
npm create vite@latest . -- --template react-ts

# Установить зависимости
npm install

# Установить дополнительные пакеты
npm install react-router lucide-react sonner
```

### Шаг 4: Скопируйте файлы из Figma Make

**Я создам для вас архив со всеми файлами!**

Напишите мне и я создам:
- `ALL_PROJECT_FILES.zip` (в виде markdown файлов)
- Каждый файл с его полным содержимым
- Инструкции по копир��ванию

### Шаг 5: Закоммитьте и запушьте

```bash
git add .
git commit -m "Initial commit - Crew Table app"
git push origin main
```
